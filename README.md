## 手机投屏电脑有几种方式？傲软投屏能做到哪些？

很多人不知道，手机投屏到电脑其实分两条路。一条是「只看」，手机画面显示在电脑上，手机还是得拿在手里操作。另一条是「反控」，手机放一边，鼠标键盘直接在电脑上控制手机里的 App。

傲软投屏把这两条路都覆盖了。

**连接方式支持两种：**

- **WiFi 无线投屏**：手机和电脑接同一个局域网，打开软件自动识别设备，点一下就连上
- **USB 有线投屏**：数据线连接，稳定性更高，适合要求低延迟的手游场景

**iOS 和安卓的差别：**

- iPhone/iPad：通过 AirPlay 协议投屏，iOS 11 以上打开控制中心点「屏幕镜像」，选择 Apowersoft 设备名称即可
- 安卓设备：通过 Chromecast 协议连接，安卓 5.0 以上均支持；USB 模式下需要先开启开发者选项中的 USB 调试

## 傲软投屏怎么用？手机投屏电脑完整步骤

### WiFi 无线投屏（安卓 & iOS 通用）

1. 在电脑上下载并安装傲软投屏桌面版（Windows/Mac 均可）
2. 在手机上安装傲软投屏 App（App Store 或应用宝搜索「傲软投屏」）
3. 确保手机和电脑连接同一个 WiFi 网络
4. 打开电脑端软件，界面会自动等待设备连接
5. 打开手机 App，点击屏幕中间的投屏按钮【M】，搜索到电脑设备名称后点击连接
6. 手机画面出现在电脑屏幕上，投屏完成

整个过程大概一分钟。第一次用可能会被防火墙拦一下，允许通过就好。

### USB 有线投屏（安卓）

1. 安卓手机进入「设置 > 开发者选项」，开启 USB 调试（不同品牌手机路径略有差异）
2. 用数据线连接手机和电脑
3. 电脑端打开傲软投屏，软件自动检测到设备后开始安装投屏 App
4. 手机上弹出授权窗口，选择「一律允许使用这台计算机进行调试」，点击确定
5. 等进度条走完，投屏成功

USB 方式延迟更低，玩手游时推荐这种。

### iPhone USB 投屏

1. 用 Lightning 或 USB-C 数据线连接 iPhone 和电脑
2. 电脑端傲软投屏自动检测，提示安装驱动时点击「立即安装」
3. 手机上出现「信任此电脑」弹窗，点信任
4. 电脑端点「已信任，下一步」，等待投屏连接成功

## 常见问题解答

**Q：傲软投屏 WiFi 投屏连接不上怎么办？**

A：最常见的原因是手机和电脑不在同一个 WiFi 下。部分路由器有「AP 隔离」设置，会阻止同网络设备间的通信，关掉这个选项就能连上。如果还不行，切换 USB 有线模式基本能解决。

**Q：安卓投屏有声音，但 iPhone 投屏没声音？**

A：iOS 的 AirPlay 投屏，声音是从手机端输出的，电脑端默认不带声音。使用傲软投屏 App 内的投屏方式（不是系统 AirPlay）可以同时传音频，投屏时在 App 里选对应选项即可。

**Q：傲软投屏支持同时连接多台设备吗？**

A：付费版（VIP 版本）支持同时将最多 4 台设备投屏到一台电脑上显示，免费版不支持这个功能。

**Q：在数码荔枝买的激活码能用在官网下载的软件上吗？**

A：可以。数码荔枝销售的是 Apowersoft 官方授权，激活码完全兼容从官网下载的版本，不存在兼容性问题。

**Q：傲软投屏支不支持退款？**
